# Text Editor

A Notion-inspired text editor built with React and Tiptap.

## Features
- Multi-page notes
- Title + body editor
- Rich text formatting (Bold, Italic, Underline)
- Bubble menu on text selection
- Persistent storage using IndexedDB
- Clean sidebar navigation

## Tech Stack
- React
- Tiptap
- IndexedDB (Dexie)
- Tailwind CSS
- Vite

## Why this project
Built to understand real-world editor architecture, state management, and UX challenges.
